# CS50 Filter (Comfortable/Generalized)

## 简介 (Introduction)
This is my C implementation of image filters for Harvard's CS50 course.
这是我在哈佛 CS50 课程中完成的图像滤镜项目 (C语言实现)。

## 特色 (Highlights)
- **Generalized Blur Algorithm:** Unlike the standard fixed 3x3 kernel, I implemented a dynamic loop structure that handles image boundaries mathematically. This allows for adjustable blur radius in the future.
- **泛化模糊算法：** 不同于标准的 3x3 固定写法，我实现了一套通用的边界处理逻辑，为未来支持可调节的模糊半径打下了基础。

## 包含的滤镜 (Filters Included)
1.  **Grayscale:** Converts image to black and white.
2.  **Sepia:** Applies an old-timey effect.
3.  **Reflection:** Flips the image horizontally.
4.  **Blur:** Box blur with edge handling.
